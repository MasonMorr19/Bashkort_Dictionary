## Overview

The Bashkir Memory Palace Enhanced Edition combines the traditional memory palace technique with advanced language learning features to help students master the Bashkir language. This application integrates Ibn Arabi's mystical framework, memory palace techniques, and anthropological pedagogy with enhanced audio dictionary features.

## Features

- **Memory Palace Navigation**: Explore Bashkir vocabulary through four symbolic locations
- **Four Birds Framework**: Learn through the spiritual framework of Eagle (Intellect), Crow (Body), Anqa (Matter), and Ringdove (Soul)
- **Advanced Audio Dictionary**: Comprehensive word lookup with audio generation
- **Spaced Repetition System (SRS)**: SM-2 algorithm for optimal review scheduling
- **Semantic Networks (BashkortNet)**: Explore cultural and linguistic connections between words
- **Mnemonic Generation**: AI-powered memory aids using the Four Birds framework
- **Sentence Builder**: Construct grammatically correct Bashkir sentences
- **Cultural Context**: Deep cultural insights into Bashkir traditions and heritage
- **Translation Services**: Real-time translation capabilities
- **Audio Analysis**: Detailed audio property analysis

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `streamlit run app.py`

## Usage

The application provides several interconnected modules:

1. **Palace Navigation**: Start your journey through the four symbolic locations
2. **Four Birds**: Understand the spiritual framework guiding your learning
3. **Sentence Builder**: Create and practice constructing Bashkir sentences
4. **Review System**: Practice vocabulary using spaced repetition
5. **BashkortNet**: Explore semantic relationships between words
6. **Cultural Context**: Learn about Bashkir cultural heritage
7. **Settings**: Customize your learning experience
8. **Audio Dictionary**: Access comprehensive word definitions and audio

## Technology Stack

- Python 3.8+
- Streamlit for the web interface
- gTTS for text-to-speech
- Deep Translator for translations
- OpenAI Whisper for speech recognition
- Transformers for NLP tasks
- Librosa for audio analysis

## Architecture

The application is organized into several modules:

- `modules/`: Core functionality modules (audio service, spaced repetition, sentence builder, etc.)
- `data/`: Vocabulary and configuration data
- `utils/`: Utility functions and helpers
- `audio_cache/`: Cached audio files for performance
